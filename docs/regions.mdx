TopK is currently available in the following regions:

| Cloud | Region | Location | Name |
|-------|---------|-----------|-----|-----|
| AWS | us-east-1 | N. Virginia | `aws-us-east-1-elastica` |
| AWS | eu-central-1 | Frankfurt | `aws-eu-central-1-monstera` |

<Info>
  To deploy TopK in your own VPC, please [contact us](/support).
</Info>
